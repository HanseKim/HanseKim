**기술스택**: React Native Expo, Node.js, Express

- 스포츠 관람을 위한 **올인원 플랫폼** (모임 생성 → 장소 예약 → 결제 → 채팅)

- 부산 지역 상권 활성화를 위한 **지역 상생 모델**
